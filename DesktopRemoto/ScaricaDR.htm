Istruzioni

È possibile scaricare il software da questo indirizzo http://www.microsoft.com/downloads/details.aspx altrimenti clicca qui .
Gli utenti di Windows XP Professional e Windows Server 2003 devono specificare la directory di installazione.

  1. Alla richiesta Selezionare il percorso per la directory contenente le pagine Web di esempio presentata dal programma di installazione, immettere C:\Windows\Web\TSWeb. Fare clic su OK.

 

  1.  Se la cartella TsWeb non esiste, viene chiesto se si desidera crearla. Fare clic su Sì.

 

Per scaricare e installare Connessione Web desktop remoto in Windows NT 4.0 o in Windows 2000:

Selezionare la lingua dall'elenco a discesa in questa pagina, quindi fare clic su Vai.

1.      Leggere il contratto di licenza. Se lo si approva, accettare i termini del contratto di licenza.

2.       Selezionare Salva l'applicazione su disco nella finestra di dialogo Download del file.

3.       Scegliere in che posizione del computer salvare il file, quindi fare clic su Salva.

4.       Portarsi nella posizione dove è stato salvato il file e fare doppio clic sul file per eseguire il programma di installazione.

5.       Alla richiesta Selezionare il percorso per la directory contenente le pagine Web di esempio presentata dal programma di installazione, immettere C:\Windows\Web\TSWeb. Fare clic su OK.
Se la cartella TsWeb non esiste, viene chiesto se si desidera crearla. Fare clic su Sì.

L'aggiornamento del software Connessione Web desktop remoto in questa pagina può essere installato su sistemi Windows 2000 e Windows NT Server 4.0 con IIS 4.0 o versioni successive.

Per scaricare e installare Connessione Web desktop remoto nelle versioni precedenti:
1. Selezionare la lingua dall'elenco a discesa in questa pagina, quindi fare clic su Vai.
2. Leggere il contratto di licenza. Se lo si approva, accettare i termini del contratto di licenza.
3. Selezionare Salva l'applicazione su disco nella finestra di dialogo Download del file.
4. Scegliere in che posizione del computer salvare il file, quindi fare clic su Salva.
5. Portarsi nella posizione dove è stato salvato il file e fare doppio clic sul file per eseguire il programma di installazione.
6. Il programma installa i file nella directory specificata. La directory di installazione predefinita è C:\Inetpub\wwwroot\Tsweb.

Come effettuare la disinstallazione

Per disinstallare Connessione Web desktop remoto:
1. Fare clic sul pulsante Start, quindi su Pannello di controllo.
2. Fare clic su Installazione applicazioni.
3. Selezionare Client Web Servizi terminal, quindi fare clic su Cambia/Rimuovi per effettuare la disinstallazione.

Per gli utenti di Windows Server 2003 non è necessario scaricare questo pacchetto. Possono aggiungerlo manualmente da Installazione applicazioni del Pannello di controllo. Il pacchetto viene offerto ai clienti Microsoft.

Dopo l'istallazione  premere qui  per scaricare le impostazioni di connessione.
